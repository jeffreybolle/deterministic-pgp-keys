# Deterministic PGP Keys
Generate PGP keys from a 12 word seed phrase

Example:
```bash
$ ./deterministic-pgp-keys --name 'Jeffrey Bolle' \
                           --email 'jeffreybolle@gmail.com' \
                           --date 2022-09-21 \
                           --generate \
                           --public-key public.asc \
                           --private-key private.asc


Seed Phrase:

   1: design     7: bubble
   2: car        8: hospital
   3: dutch      9: muffin
   4: struggle  10: earn
   5: hello     11: half
   6: pluck     12: best

```
